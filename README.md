# ⚙️ Instalasi

Clone repository

```bash
git clone https://github.com/rayhan204/Portofolio-Website-Rayhan.git
```

Masuk ke folder project

```bash
cd Portofolio-Website-Rayhan
```

Install dependency

```bash
npm install
```

Menjalankan project

```bash
npm run dev
```

Build Production

```bash
npm run build
```
